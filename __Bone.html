service

Web Design

Web Site ・ UX Design

HTML5＋CSS3で、メンテナンス性の高い、SEOにも適したWebサイトを制作いたします。

Webサイトデザインは、クライアント様の成果につながる非常に重要なものです。 弊社ではデザイン制作を行う前に、ヒアリングにてご要望の特徴や魅力を引き出し、ユーザーインターフェイスを重視した設計とデザインにより、メインコンセプトが伝わりやすく、触りやすいWebサイトをプロデュース致します。

Webサイトと一口に言っても目的や用途に合わせて様々な形がありますように、

クライアント様の目的や予算に合わせたフレキシブルな形でご提案させて頂きます。

開設後の定期的な更新など、アフターサポートや保守、ランディングページ、キャンペーンサイトの制作も承ります。

Responsive Design

Mobile ・ Multiple Device

新規制作のレシポンシブWebデザインのほか、既存のホームページをスマートフォン、タブレット端末にも適したモバイルフレンドリーな表示に切り替えるサポートをいたします。

スマホユーザやタブレットユーザが増加していく中、どのデバイスからも読みやすく操作性のよいWebサイトを提供することは、クライアント様にとってもユーザ満足度の向上になると考えています。

レスポンシブWebサイトはデバイスの画面サイズに合わせたレイアウト表示をするので、新しいデバイスが増えても対応可能。また、デバイス毎に専用サイトを制作する必要がなく、比較的コスト面でもお得です。

CMS

WordPress ・ PHP ・ CGI

日々のサイト更新作業の軽減やブログでの情報発信など、クライアント様の用途・目的に合わせたCMS(コンテンツ・マネジメント・システム)をご提案、実装致します。

HTMLやCSSの知識が無くても、管理画面からテキストの修正や写真の差し替えなどカンタンにWebサイトの更新が行えます。

サイト丸ごとCMS化したい、サイトの一部だけ自社更新したい等、ご要望ベースで個々にご対応させて頂きます。更新費用や時間の削減、場所を選ばずリアルタイムでの情報発信など、サイトを運営する側にとってはCMS導入はさまざまなメリットがあります。

WordPressによるサイト構築やPHP/CGI等を中心に、更新・拡張に優れたCMSサイトを制作。

Movable TypeやJimdoを利用した動的サイトの制作、ブログのカスタマイズのご依頼もお気軽にお問い合わせください。

Online Shop

EC Site ・ Store Design

「カラーミーショップ」や「EC-CUBE」等によるオンラインショッピングサイトのデザイン・カスタマイズを中心に、クライアント様のニーズに合わせたECサイトを制作致します。

動機や目的、条件を考え「買う人の目線」を重視した商品の見せ方や導線を設計します。

出店準備からのプロデュースや既存ショップのリニューアル以外にも、フリーページを利用したランディングページの制作、特集ページやキャンペーンページの制作もご相談ください。

各種ASPやショッピングモールの特徴を把握し、効果を生むショッピングサイトの制作を行います。

workflow

コンサルティング

コンタクトフォームよりご依頼頂いた内容等を、メールまたは直接お会いしてお打ち合わせさせて頂きます。お客様と制作スタッフとの間で共有の完成イメージをしっかりと作ることが重要です。 サイトのコンセプト、デザイン、運営方法、制作スケジュール等をお伺いして、企画書と御見積書を作成致します。

制作

企画書と御見積書をお客様にご確認頂いた後、制作に入ります。その際に必要なデータ(原稿、挿入写真等)をご用意頂き、数日後にサイトデザインをご提案させて頂きます。 全体のデザインが決定致しましたらページ制作に入ります。デザインの統一性、ユーザビリティー等を全体に配慮しながら、制作進行致します。

校正

テストサーバーに全データを仮アップし、デザイン・レイアウト・文字校正等をご確認して頂きます。修正作業を行い、最終確認をして頂きます。

納品

全データが完成致しましたら、サーバーへアップロードし、インターネット上への公開まですべてサポートさせて頂きます。データでの納品等、ご要望に応じた納品形態も可能です。

メンテナンス

ホームページ更新・管理も承ります。修正や削除ヶ所などをメールまたはFAXでお送り頂ければ、基本的に3営業日以内に修正させて頂きます。

＊WEB制作の基本的な流れとなりますが、急ぎの制作依頼や内容によっては、この限りではありません。 お気軽にお問い合わせください。
